# SinarCare
SinarCare is a project created by KambingByte for Project 2030: MyAI Future Hackathon organised by Universiti Teknologi Malaysia 2026. 

SinarCare is our take on solving Grand Challenge Track 3: Vital Signs.

We aim to build an application for elderly where they can have a sense of independence and is also easy for their caregivers or family members to monitor. The main objective of SinarCare is to combat the overcrowding of emergency department in hospitals and help elderly become more independent of their health. The app mainly focuses on main page use without any technical jargons.
The app has a simple interface with large buttons and clear navigation. It also uses soft colours that complement the features of the app. This app helps users with medication reminders, set up appointments, an SOS button, and others. 

We present to you, SinarCare, _"Assistance for Our Loved Ones"_.
